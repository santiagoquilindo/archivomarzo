Como usar el indice local

El panel administra un indice documental local. Primero defines fuentes, luego sincronizas y despues consultas el resultado ya indexado.

  1. 1. Agregar carpeta Registra una o varias carpetas raiz para definir donde leer documentos.
  2. 2. Ejecutar indexacion Procesa solo carpetas activas y actualiza el indice disponible para buscar.
  3. 3. Buscar documentos Consulta unicamente documentos indexados en el estado actual del indice.
  4. 4. Limpiar o reemplazar indice Si cambias carpetas o quieres reiniciar el indice, limpia y vuelve a indexar.

Carpetas raiz documentales

Administra las ubicaciones base del sistema y controla cuales participan en los procesos documentales. Si no hay carpetas agregadas, no habra documentos para indexar.
